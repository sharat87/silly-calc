This is an annoyingly simple notepad style calculator. It is currently a work-in-progress, but I'm already using it for my work and study. If you find any problem or a missing function/functionality, feel free to drop me a line.

All calculations are done locally and all data is saved locally in your browser. There is currently no server communication.
